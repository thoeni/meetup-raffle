## Meetup Raffle

This CLI allows to query Meetup.com participants list for GLUG meetups and pick a participant.

### Pre-requisite:
- Go installed
- iTerm2

### How to call it:
`go run main.go <meetupURL>`

Example:
```
go run main.go https://www.meetup.com/Go-London-User-Group/events/248310043/
```
